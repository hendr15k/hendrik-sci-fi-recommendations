# Hendrik's Sci-Fi Empfehlungen 📚🚀

Kuratierte Science-Fiction Buchempfehlungen — als statische GitHub-Page.

**[Live ansehen →](https://hendr15k.github.io/hendrik-sci-fi-recommendations/)**

## Was ist das?

Eine persönliche Lese-Liste mit ~70 Science-Fiction-Titeln, aufbereitet als durchsuchbare, filterbare HTML-Seite. Kein Framework, kein Build-Step — pures HTML/CSS/JS, blitzschnell und überall lesbar.

## Features

- 📚 **~70 Sci-Fi Titel** — persönlich kuratiert
- 🔍 **Suche** — Nach Titel oder Autor filtern
- 📊 **Statistiken** — Anzahl gelesener Bücher, Seiten, durchschnittliche Bewertung
- 👍 **Ranking** — Sortiert nach persönlicher Bewertung
- 📱 **Responsive** — Funktioniert auf Desktop und Handy
- ⚡ **Blitzschnell** — Reines HTML/CSS/JS, kein Framework

## Deployment

Push to `main` → automatisch live auf GitHub Pages.
